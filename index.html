  1. 고치거나 추가한 소스코드
  • 데이터분석 특성상 전체 주제를 바꾸어 고치고 분석하였습니다.
  • https://www.kaggle.com/zusmani/us-mass-shootings-last-50-years kaggle에서 1966~2017, 2017~2021 미국 총기난사 데이터를 csv 형태로 불러온다.
  • pandas, matplotlib, numpy 모듈을 사용한다.
  • 미국 총기난사 2017~2021 데이터 분석

-원본코드와 동일하게 loc[]을 사용하여 각각의 데이터 혹은 범위를 출력한다.

-추가한코드: matplotlib을 사용하여 데이터를 막대그래프로 시각화한다.

                   numpy를 사용하여 주어진 범위와 간격에따른 균일한 값을 생성한다.

                   print(shootings['State'].value_counts())하여 정확한 사건수를 파악한다.

                   3개 주를 골라 그에 따른 사건수를 막대그래프로 나타낸다.

                     -state = ['Illinois', 'California', 'New York']

                     -values = [214, 189, 83]

                   bar()를 사용하여 막대그래프의 색상,너비,위치,두께등을 설정한다.

                   xticks()에 x와 state를 입력하여 x축에 state값을 표시한다.

  • 미국 총기난사 1966~2017 데이터 분석

-추가한코드: matplotlib에서 plot(kind = ‘pie’)를 사용하여 가해자 성별 비율을

                   원 그래프로 나타낸다.

                   autopct = '%.1f%%'로 소숫점 첫번째 자리까지 표현한다.

                  value_counts()로 각 성별에 따른 가해자수를 파악한다.

                  여성, 남성, 알 수 없음으로 구분한다.

                  구분을 위해 데이터F/M은 각 여성과 남성에 더해준다.

                  -gender = ['Female','Male','Unknown']

                  -values = [26, 313, 5]

                 explode = [0.2,0,0.2]는 부분 확대를 나타낸다.

                 pie()에 색상, 확대, 소숫점 첫째 자리에서 반올림한 값을 추가한다.
